### 4-4: Quoted attributes

**Problem**: `get_attributes` uses `text.split()`, which breaks attribute values with spaces (e.g., `href="hello world.html"`). Also, `parse()` treats any `>` as tag end, even inside quoted values.

**Hint from book**: The lexer is a 2-state FSM (`in_tag`). Needs more states to track quoted context.

**What needs to change**:
- `get_attributes()`: rewrite as character-by-character tokenizer that respects quotes
- `parse()`: add `in_quote` state so `>` inside quotes doesn't end the tag
